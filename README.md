# Tibetan National Sports Association

## How to run this project locally

## step 1:

- go to terminal or command prompt
- choose the directory where you want to keep the project
  eg. cd Desktop

## step 2:

git clone git@github.com:Rojlap107/TNSA.git

## step 3:

cd Desktop
cd TNSA

## step 4:

- rm -rf node_modules package-lock.json

## step 5:

- npm install

## step 6:

- npm run dev
